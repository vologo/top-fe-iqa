## 项目目录

本项目是笔者总结整理记录的`大前端`的技能点，重查漏补缺，十分必要。以下是项目的目录（更新中…）

```sh
├─async                     // 异步编程
├─css                       // Css基础
├─handwriting               // 手写代码
│  ├─bind-apply-call            // bind apply call
│  ├─debounceAndThrottle        // 防抖节流
│  └─deepClone                  // 深拷贝
├─html                      // html大全
└─javaScript                // javascript 查漏
    ├─array-api                 // 数组API
    └─obj-api                   // 对象API
```
