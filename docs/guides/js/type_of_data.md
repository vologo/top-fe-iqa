---
title: JavaScript的数据类型有哪些,存储有什么区别？
---

js 中的数据类型大体分为两大类：一是基本的数据类型（简单数据类型、原始类型）二是复杂数据类型

| Tables   |                    desc                     |      other |
| -------- | :-----------------------------------------: | ---------: |
| 原始类型 | Number String Null Undefined Symbol Boolean |     Symbol |
| 复杂类型 |                   Object                    | 存储的引用 |
