## GET与Post的区别

### get
用于向服务器查询某些信息。必要时，需要在GET请求的URL后面添加查询字符串参数
```js
https://jsonplaceholder.typicode.com/posts?_offset=1&_limit=2
```
### post

用于向服务器发送应该保存的数据。每个POST请求都应该在请求体中携带提交的数据，而GET请求则不然
